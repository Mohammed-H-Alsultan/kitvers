# Kitvers (WIP)

Kitvers is a desktop tool to create, manage, and evolve web project stacks with one click.

## Features (WIP)

- Create Next.js or Vite projects
- Select packs (Tailwind, daisyUI, icons, charts)
- Open existing projects
- Add packs to existing projects
- One-click dependency updates
- Optional git safety backup

## Tech Stack

- Tauri
- React (Vite)
